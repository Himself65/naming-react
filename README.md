# 点名器

> 上课随机点名学生回答问题

## 使用方法
1. 通过下载[发布版](https://github.com/Himself65/naming/releases)

2. 通过自行打包

```bash
git clone https://github.com/Himself65/naming.git
```

（**注意**）必须修改[此文件](src/people.json)

```bash
# using npm
npm install
npm run build

# or you can use yarn
yarn install
yarn run build
```
